## 초기 프로젝트 설치

```bash
$ npm install
```

## 프로젝트 실행

```bash
# 테스트 모드
$ npm run start

# 디버그 모드
$ npm run start:dev

# 운영 모드
$ npm run start:prod
```

## 테스트

```bash
# 유닛 단위 테스트
$ npm run test

# End-To-End 테스트
$ npm run test:e2e

# 테스트 커버리지
$ npm run test:cov
```
